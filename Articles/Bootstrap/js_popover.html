Позиционирование Popovers с помощью JavaScript

Для позиционирования всплывающей панели с помощью JavaScript используется параметр метода popover() placement, который может принимать следующие значения: 'top' (вверх), 'bottom' (вниз), 'left' (влево), 'right' (вправо).

Универсальные классы

Универсальные классы можно применять практически ко всем элементам Bootstrap

*-lg, .*-sm, .*-xs - большой, средний и маленький размеры элемента, например btn-lg, .btn-sm, или .btn-xs

*-primary, *-info, *-success, *-warning, *-danger, *-link - цвета

*-block, *-inline, - элемент превращается в блочный, строчный

*-disabled - внешний вид становится неактивным. Что бы элемент лишился функциональности, нужно записать "href='#'"

*-justified - применяется только к ‹a› - растягивает элемент по ширине
