Bootstrap 3 - Введение

Один из самых популярных фреймворков для разработки адаптивных и мобильных web-проектов. Bootstrap использует самые современные технологии CSS и HTML в результате чего он интуитивно прост и в тоже время мощен. При его использовании повышается скорость и облегчается разработка web-приложений.

Характерные черты Bootstrap 3

  • Высокая скорость разработки. Платформа Twitter Bootstrap позволяет более быстро и качественно создать front-end составляющие сайтов по сравнению с их созданием на "чистом" CSS и JavaScript. Это достигается за счет использования готовых классов и компонентов Twitter Bootstrap.
  • Гармоничный дизайн. Классы и компоненты платформы Twitter Bootstrap выполнены в едином стиле и очень хорошо согласуются между собой. В итоге дизайн сайта, выполненный на основе Twitter Bootstrap, выглядит стильно и привлекательно.
  • Простота в использовании. Платформа Twitter Bootstrap 3 не требует хороших знаний HTML, CSS, JavaScript и jQuery. Для работы с платформой достаточно иметь минимальные знания по технологиям HTML и CSS. В большинстве случаев основная задача при разработке веб-страниц будет сводиться к использованию готовых фрагментов кода (компонентов) и классов Twitter Bootstrap 3. Даже знания языка JavaScript и библиотеки jQuery не требуется, т.к. настраивать компоненты и логику их работы можно с помощью атрибутов data-.
  • Совместимость с браузерами. Данная платформа адаптирована для работы со всеми современными браузерами: Mozilla Firefox, Google Chrome, Safari, Internet Explorer и Opera.
  • Открытость и бесплатность программного обеспечения. Платформа Twitter Bootstrap 3 является бесплатной как для личного так и для коммерческого использования. Кроме этого, она имеет еще открытые исходные коды, т.е. её код доступен для просмотра, изучения и изменения.
  • В дополнение к CSS, Bootstrap включает в себя поддержку двух самых популярных CSS препроцессоров, Less и Sass.
  • Bootstrap легко и эффективно масштабирует ваш проект с одной базой кода, от телефонов и планшетов до настольных компьютеров.

Bootstrap 3 с ориентацией на мобильные устройства

Самое главное изменение в новой версии Bootstrap это поддержка адаптивного дизайна, но самое интересное заключается в том, что сам этот модуль был удален. Вместо этого, новый Bootstrap на уровне ядра поддерживает адаптивность.

Третья версия фреймворка выпущена под девизом «Mobile First», что означает приоритет на адаптацию под мобильные устройства. Практически весь продукт был переделан с нуля, чтобы внедрить поддержку низких разрешений экрана и масштабирования.

Во внешнем виде и ощущениях вы также можете заметить изменения. Например, стиль оформления стал плоским, но имеется возможность вернуться к внешнему виду версии 2.0. В дополнение к этому иконки из изображений перевели в шрифты, что положительно сказывается, когда необходимо использовать различные размеры и цвета.

Описание платформы Twitter Bootstrap 3

Платформа Twitter Bootstrap 3 состоит из:

  • системы сеток, которая выполнена с помощью классов CSS и предназначена для разработки адаптивной разметки веб-страниц.
  • общих классов CSS, предназначенных для оформления большинства элементов HTML и выполнения вспомогательных операций.
  • различных компонентов пользовательского интерфейса веб-страниц, реализованных на HTML, CSS и JavaScript (jQuery).

Всё это представляет собой не просто "кучку" не связанных между собой компонентов и классов CSS, а действительно платформу, где всё продумано до мелочей, с помощью которой приятно создавать интерфейсы веб-сайтов.

На файловом уровне платформа Twitter Bootstrap 3 состоит из CSS, шрифтов (иконок) и скриптов JavaScript.
